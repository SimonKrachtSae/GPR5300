Welcome!

When starting the application for the first time, pls enter the scene view first.
Here you will be presented 3 Shapes, displaying a "Rough/Matt/Glossy" Szenario:
1) Plane = Rough
2) Cube = Matt
3) Pyramid = Glossy
Next, you can head back to the start menu. From there, navigate to the Object Editor.
Here you can select any of the three objects and change ist settings. After you have
adjusted all settings to your liking, head back to the scene view and check out your
newly customized objects next to each other!
When you are done with using the application, once again, navigate to the start menu.
Here you can press the "Quit" button, in order to quit the application.
